# 🏏 Cricket Data Analysis using SQL

## 📌 Project Overview

This project is a **Cricket Data Analysis system** built using MySQL.
It analyzes player performance, match statistics, and key metrics using structured queries.

---

## 🗄️ Database Details

* Database Name: `cricket_stats`
* Tables:

  * batting_stats
  * bowling_stats
  * fielding_stats
  * players

---

## 📊 Key Insights

* Player performance tracking
* Strike rate analysis
* Match-wise statistics
* Data-driven insights using SQL queries
